=encoding utf8

=head1 NAME

pgm2txt - convert PGM image files to CAPH text input files

=head1 SYNOPSIS

pgm2txt [-abbrev] in_file out_file

=head1 DESCRIPTION

This program can be used to convert PGM files to text files which can be read by the CAPH
interpreter and the testbench generated by the SystemC backend.

=head1 ARGUMENTS

=over

=item in_file

name of the input file 

=item out_file

name of the output file 

=back

=head1 OPTIONS

=over

=item -abbrev

use abbreviated syntax for tokens in the ouput file (C<v> for C<Data v>, C<<> for C<SoS> and C<>> for C<EoS>)        

=back

=head1 EXIT STATUS

The program returns a zero exist status if it succeeds and a non zero value in case of failure. In
the latter case, an error message is printed to stderr.
